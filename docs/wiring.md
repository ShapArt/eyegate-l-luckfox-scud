# Схема подключений (словесное описание)

## Управление замками дверей

Каждый замок питается от отдельного блока 12 В через реле‑модуль:

- **Дверь 1**:
  - +12 В → COM реле 1,
  - NO реле 1 → + замка двери 1,
  - минус замка → GND блока питания.

- **Дверь 2**:
  - +12 В → COM реле 2,
  - NO реле 2 → + замка двери 2,
  - минус замка → GND блока питания.

Катушки реле управляются от GPIO платы через транзистор/оптрон:

- Вход IN1 реле 1 → GPIO `DOOR1_LOCK` (через драйвер),
- Вход IN2 реле 2 → GPIO `DOOR2_LOCK`.

### Принятая привязка GPIO (аппаратный режим прототипа)

В текущей реализации (см. `server/deps.py`) используется следующая привязка сигналов к GPIO:

| Сигнал | Назначение | GPIO |
|---|---|---:|
| `DOOR1_LOCK` | Управление замком Door1 (реле/драйвер) | 41 |
| `DOOR2_LOCK` | Управление замком Door2 (реле/драйвер) | 42 |
| `DOOR1_CLOSED` | Датчик закрытия Door1 (концевик/геркон) | 40 |
| `DOOR2_CLOSED` | Датчик закрытия Door2 (концевик/геркон) | 39 |
| `ALARM_OUT` | Управление сиреной/лампой (реле/драйвер) | 36 |

## Концевые датчики дверей

Для каждого геркона (NC/NO) делаем цепочку на вход GPIO с подтяжкой:

- `DOOR1_CLOSED`:
  - один контакт геркона → GND,
  - второй контакт → GPIO с включённой внутренней подтяжкой к 3.3 В
    (или внешним резистором ~10 кОм).
- `DOOR2_CLOSED` — аналогично.

Логика в коде (`closed_active_high=False`):
замкнуто = лог.0 = «дверь закрыта».

## Сигнализация (сирена/лампа)

- +12 В → COM реле ALARM,
- NO реле → + сирены/лампы,
- минус сирены/лампы → GND блока питания.

GPIO `ALARM_OUT` управляет входом IN_ALARM реле (через драйвер).

## Считыватель карты

В зависимости от выбранной модели:

- USB‑NFC: подключается к USB порту платы, в Linux виден как `/dev/hidraw*`
  или `/dev/ttyUSB*` (обработка — отдельный модуль/демон).

- Wiegand: линии D0/D1 подключаются к небольшому микроконтроллеру/конвертеру,
  который по UART передаёт номер карты на Luckfox.

В учебной версии логика карты реализована как вызов `card_presented(card_id)`,
например, из CLI/теста или отдельного скрипта.
